# Dropwizard-guicey

Core dropwizard-guicey library.


Maven:

```xml
<dependency>
  <groupId>ru.vyarus</groupId>
  <artifactId>dropwizard-guicey</artifactId>
  <version>{ guicey.version }</version>
</dependency>
```

Gradle:

```groovy
implementation 'ru.vyarus:dropwizard-guicey:{ guicey.version }'
```
